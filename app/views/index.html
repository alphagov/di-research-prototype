Baseline journey

Summary

This is the baselined version of the {{ serviceName }} journey as of December 2024, when the driving licence authoritative source check changes were added (login to app version).

This version assumes a medium confidence requirement and defaults to Experian KBVs.

Notable differences from live
  • Updated triage flow based on the introduction of the new strategic app, as well as device sniffing software (June 24)
  • A revised version of the address CRI which has not been put live yet, but doesn't alter the outcome of the address journey (Nov 23)
  • Updated KBV start page for the introduction of DWP KBVs (Sept 23)
  • Changes to driving licence authority page in line with Low Confidence initiative updates (Aug 24)
